# 📖 Project Overview

Expanding into new markets is one of the highest-risk strategic decisions a business can make.

This project develops a data-driven framework for evaluating market attractiveness using customer analytics, geographic analysis, financial scenario modelling, and competitive assessment.

The project helps executives determine **where** to expand, **when** to expand, and **how** to prioritize investment.

---

# 🎯 Business Problem

The company lacks a structured methodology for selecting the most attractive Brazilian markets for expansion.

Questions addressed include:

- Which state should be entered first?
- Which market offers the highest ROI?
- Which customers should be targeted?
- How should investment be phased?

# 📈 Key Insights

- Goiás provides the highest immediate expansion opportunity.
- Minas Gerais offers the largest long-term market.
- Rio Grande do Sul provides lower-risk diversification.
- Customer quality differs significantly between states.
- Expansion decisions should prioritize customer value rather than customer volume.

---

# 💼 Business Recommendations

✔ Enter Goiás first

✔ Expand into Minas Gerais during second phase

✔ Target Loyal and Champion customers

✔ Prioritize high ROI regions

✔ Monitor competition continuously
